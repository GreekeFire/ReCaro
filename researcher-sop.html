STEP 03
Find the Highest Price
Some products have multiple variants. You must check all of them.

Some products on Shopee come in different versions — different sizes, colours, or sold as a bundle. Each version may have a different price. You must find and record the highest price.

1

On the product page, look for any options to select — such as size, colour, quantity, or bundle.

📷
images/image5.png
Screenshot showing product size/colour variant buttons on Shopee
2

Click through every single option. Watch the price displayed on the page change each time.

📷
images/image6.png
Screenshot showing how the price updates as you click different variants
3

Write down the highest price you saw across all options.

Example A storage box comes in Small ($8.90), Medium ($12.90), and Large ($15.90). You must write down $15.90 — the highest price. Do NOT write the cheapest price. Do NOT write the default price without checking all options first.
✓
Only one version If there is only one version of the product with one price, just write down that price and move on.
STEP 04
Save the Product Images
Minimum 5 images. Maximum 10. Cover image must be clean.
✗
Fewer than 5 images If the product on Shopee has fewer than 5 images, do not use that product. Go back to Step 1 and find a different product.
// Which images to save
1

Save product images from the Shopee listing. Save a minimum of 5 and a maximum of 10. If there are more than 10, pick the 10 clearest ones.

2

Do not save lifestyle photos that show people using the product. Only save photos of the product itself.

// Choosing the cover image

The cover image is the first image — the one buyers see before they click. This is the most important image. Choose carefully.

Good cover image
  • Clear photo of just the product
  • Clean background — white or plain colour
  • No text or very little text
  • No brand name or logo visible
  • No watermarks
Bad cover image
  • Brand name or logo clearly visible
  • Large text or promotional banners
  • Watermarks anywhere on the image
  • Blurry or dark image
  • Photos showing people using the product

STEP 05
Write the Product Description
Use AI to write this. Do the description before the title.
⚠
Description before title You must write the description BEFORE the title. Do not skip ahead. Always do the description first — you will use it to create the title in Step 6.
1

On the Shopee product page, scroll down to find the product description. This is where the seller has written about the product.

📷
images/image11.png
Screenshot showing where to find the product description on a Shopee listing page
2

Select all the text in the product description. Copy it with Ctrl+C / Cmd+C.

📷
images/image12.png
Screenshot showing the description text selected and being copied
3

Open ChatGPT or your approved AI tool in a new tab.

4

Copy the prompt below using the Copy button.

// Prompt — Write Description
You are an expert Carousell listing copywriter. Write a product description that converts browsers to buyers. STRUCTURE (use this exact format): **[Product Name - Hook]** [One sentence describing the product and main benefit] **Why You'll Love It:** • [Benefit 1 - what it does for them] • [Benefit 2 - emotional/satisfaction] • [Benefit 3 - practical/specific] • [Benefit 4 - quality/convenience] **Perfect For:** [Specific use cases - who needs this and where they use it] **What's Included:** [List exactly what they get] **Specifications:** • Material: [material] • Dimensions: [L x W x H in cm] • Weight: [in kg or grams] • Color: [available colors] **Why Buy From Me:** ✓ Fast Singapore shipping (2-3 days) ✓ Quality checked before sending ✓ Secure packaging ✓ Responsive customer service **Note:** [Stock status or any important info] --- 🎁 DM me for bulk discounts or questions! TONE GUIDELINES: - Friendly but professional - Focus on benefits, not just features - Use "you" and "your" to connect with buyer - Be specific — NO vague claims like "high quality" - Total length: 100-150 words AVOID: - "Amazing," "incredible," "best" - All caps words - Multiple exclamation marks (!!!) - Copy-paste from manufacturer - Mentioning Shopee, Amazon, or other platforms - Including any brand names
5

In the AI chat, paste the prompt first. Then on a new line, paste the Shopee product description you copied. Then press Enter.

📷
images/image13.png
Screenshot of ChatGPT showing the prompt and Shopee description pasted together
6

Wait for the AI to finish. Copy the full description it gives you and save it in your notes.

✗
Check before saving If the AI description mentions Shopee, Amazon, Lazada, or any other platform — delete that sentence. If the description includes a brand name — replace it with a general product name. Example: change "Ikea stackable box" to "stackable storage box".
STEP 06
Write the Listing Title
Use the description you just wrote as input for the AI.
1

Go back to your AI tool. You will continue in the same chat or start a new message.

2

Copy the prompt below using the Copy button.

// Prompt — Write Title
You are an expert Carousell listing copywriter. Create an SEO-optimized title (maximum 225 characters) for this product. CAROUSELL TITLE BEST PRACTICES: - Front-load important keywords (first 50 chars most visible) - Include: [Product Name] | [Key Feature] | [Use Case/Category] - Use pipes (|) or dashes (-) to separate keywords - Include synonyms and related search terms - Capitalize major words (Title Case) - NO special characters: ! @ # $ % * & - NO emojis in title - NO vague words: "nice," "good," "cheap," "best" - NO brand names WHAT BUYERS SEARCH FOR ON CAROUSELL: - Style descriptors: "Korean," "Japanese," "Minimalist," "Nordic" - Use cases: "Home," "Office," "Travel," "Gift" - Qualities: "Portable," "Foldable," "Adjustable," "Waterproof" - Materials: "Stainless," "Bamboo," "Silicone," "Acrylic" FORMAT: Product Name | Key Feature | Use Case | Secondary Keywords IMPORTANT: 1. Character count must be under 225 2. Include words buyers actually search for 3. No filler words like "Brand New" 4. Make it specific, not generic
3

In the AI chat, paste the prompt first. Then on a new line, paste the description you created in Step 5. Then press Enter.

📷
images/image14.png
Screenshot of ChatGPT generating a title from the description
4

Copy the title the AI gives you and save it in your notes.

// Check the title before saving
Title must have all of these
  • Under 225 characters
  • No brand names
  • No special characters: ! @ # $ % * &
  • No emojis
  • No vague words: "nice," "good," "cheap," "best"
If the title fails any check
  • Title over 225 chars → type "Please shorten this to under 225 characters" and copy the new one
  • Has brand name → ask AI to remove it and replace with a generic term
  • Has special chars → ask AI to remove them
STEP 07
Create Your Google Drive Folder
One folder per product. Name it exactly as shown below.
1

Open Google Drive and create a new folder.

You should have a main folder for your VA code already. Create this new product folder inside it.

2

Name the folder using this exact format: [Your VA Code] - [Product Name]

Examples RVA001 - Bamboo Storage Box
RVA001 - Silicone Kitchen Spatula Set
RVA001 - Foldable Car Boot Organiser
3

Upload all your product images into the folder (5–10 images, cleaned cover image first).

4

Set the folder's sharing to "Anyone with the link".

Right-click the folder → Share → Change to "Anyone with the link" → Copy link

5

Copy the folder link. You will need it when you submit the form.

✗
Folder must be shared If you forget to set sharing to "Anyone with the link," the reviewer will not be able to open it. Your submission will be rejected and logged against your VA code.
⚠
Folder link, not file link Copy the link to the folder, not to an individual image inside it. The folder link starts with https://drive.google.com/drive/folders/
STEP 08
Submit the Form
Fill in every field. Do not leave anything blank.

Open your personal submission form link (the one you were given when you joined). Fill in all fields using the information you collected in Steps 1–7.

1

Product Title — paste the title from Step 6

Must be under 225 characters

2

Product Description — paste the description from Step 5

Must be at least 100 characters

3

Shopee Link — paste the URL from Step 2

Must start with https://shopee.sg/

4

Source Cost — enter the highest price from Step 3

Numbers only, no $ sign

5

Drive Folder Link — paste the Google Drive folder link from Step 7

Must be a drive.google.com link, must be set to "Anyone with the link"

6

Click Submit. Wait for the confirmation message before moving on to your next product.

✓
On successful submission The form will show you the calculated sell price and confirm the submission was received. If you see an error message instead, read it carefully and fix the issue before resubmitting.
ℹ
Your VA code is pre-filled Your VA code is already filled in on your form. Do not change it. Do not use anyone else's form.
